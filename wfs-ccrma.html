Frugal Wave Field Synthesis System

Overview

The Frugal Wave Field Synthesis (WFS) system is a cost‑effective, FPGA‑driven spatial audio platform designed to make real‑time WFS accessible and reproducible. This page provides a complete walkthrough of the system architecture, hardware, software, and usage instructions.

Hardware Architecture

At the core of the system is a custom PCB interfaced directly with an FPGA. The board manages 32 amplified loudspeaker channels with ultra‑low latency. The design eliminates expensive multi‑device setups typically required for WFS.

  • 32 amplified channels
  • Direct FPGA connection via TDM / custom bus
  • Low‑noise power distribution
  • Stackable for expanded arrays

Software Pipeline

The DSP is generated using the Syfala compiler, allowing real‑time synthesis of WFS algorithms directly from Faust or C++.

  • Source encoding (virtual point sources)
  • Delay‑based or filtered WFS formulations
  • Real‑time monitoring through USB‑hosted GUI

User Guide

1. Powering the System

Use a 24V power supply capable of delivering at least 4A for a single 32‑channel unit.

2. Connecting the FPGA

Attach the board to your FPGA using the provided ribbon cables or TDM interface. Ensure clock synchronization is enabled in the bitstream.

3. Audio Input

Depending on your FPGA setup, audio can be streamed via:

  • Analog inputs
  • USB audio interface
  • Embedded Linux streaming

4. Configuring a WFS Scene

Use the GUI or control board to set virtual source positions (x, y). The system updates rendering instantly.

5. Expanding the Array

Multiple boards may be chained to increase the number of speakers. Each board automatically aligns timing based on its identifier.
